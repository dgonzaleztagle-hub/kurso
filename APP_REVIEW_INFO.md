# App Review Information

Usar este documento para cargar la seccion `App Review Information` en App Store Connect.

## Review Notes sugeridas

Kurso es una aplicacion de gestion financiera y administrativa para cursos y comunidades escolares.

La app permite:

- inicio de sesion por usuario invitado o creado por administradores del tenant
- gestion de pagos, deudas y rendiciones
- consulta de movimientos y portal publico asociado

La app requiere autenticacion para la mayor parte de sus funciones. Para revision de Apple se debe entregar:

- un usuario de prueba valido
- password temporal
- nombre del tenant o curso de prueba
- instrucciones breves de ingreso
